# Backend

This directory contains the backend code for the CV Maker application. The backend is built using Node.js and NestJS, and it handles API requests, data storage, and business logic for generating and managing CVs.

## Installation

```bash
pnpm install
```

After installing, run the following command to install Puppeteer browsers:

```bash
npx puppeteer browsers install chrome
```

Now you are ready to run the backend server.

## Running the Application

### Development Mode

```bash
pnpm dev
```

### Production Build

```bash
pnpm build && pnpm start
```
